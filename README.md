# Gemini Quizify

Gemini Quizify is an AI-powered tool that generates multiple-choice quizzes from user-uploaded PDFs. It uses **Google Gemini** and **Vertex AI API** for document embeddings, ensuring quizzes stay true to the source content. The project also leverages:

- **Google Service Account** for authentication
- **Langchain** for structured output parsing
- **PDF Loader** for document ingestion
- **Chroma** for embedding persistence
- **Streamlit** for an interactive UI
